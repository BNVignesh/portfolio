Leetcode Friends Info

I developed a browser extension that allows users to easily view their friend's LeetCode profile statistics with a single click. The extension uses Local Storage to save usernames and automatically fetch the latest profile data, eliminating the need to re-enter them every time. To ensure compatibility, I created and maintained separate branches for Chromium-based browsers and Firefox forks. After development and testing, I successfully published and got the extension verified on addons.mozilla.org, making it publicly available for use.

Online Notes-app

I built a full-stack web application that allows users to securely create, store, and manage their notes with a simple login. Users can create, edit, and delete notes, as well as add tags for better organization. To make note retrieval effortless, I implemented a powerful search feature that quickly locates the required notes. The application is secured with JWT authentication, significantly reducing security vulnerabilities, and features a clean, user-friendly interface built with Tailwind CSS. On the backend, I leveraged MongoDB for fast and reliable data storage, resulting in enhanced data access speed and performance.

Disease-Tracker
